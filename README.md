# ✅ To-Do List

Aplicação de lista de tarefas (To-Do List) feita com **React** e estilizada com **Bootstrap**.

## 🚀 Tecnologias

- [React](https://react.dev/)
- [Bootstrap](https://getbootstrap.com/)

## 📋 Pré-requisitos

- [Node.js](https://nodejs.org/) instalado (recomendado v18+)

## ⚙️ Instalação

```bash
# Clone o repositório
git clone https://github.com/thbbsa/ToDoList.git

# Entre na pasta do app
cd ToDoList/my-app

# Instale as dependências
npm install
```

## ▶️ Como rodar

```bash
npm start
```

O app vai abrir automaticamente em `http://localhost:3000`.

## ✨ Funcionalidades

- Adicionar tarefas
- Marcar tarefas como concluídas
- Remover tarefas

## 📝 Sobre

Projeto feito como exercício de estudo de React.
